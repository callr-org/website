Installing Bioconductor packages

source('http://callr.org/biocLite#limma,edgeR')

This is a one-line call identical to

source('http://bioconductor.org/biocLite.R')
biocLite(c('limma', 'edgeR'))

The biocLite command:

  • Works in a freshly installed R session.
  • Requires no prior package installations.
  • Will always install requested packages regardless whether they are already installed or not. To avoid this, use the install command.
